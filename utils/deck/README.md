Utils and styles for the [DeckDeckGo] applications.

## Table of contents

## Getting Started

If you would like to install and compile locally this project, proceed as following:

```
git clone https://github.com/deckgo/deckdeckgo
cd deckdeckgo/utils/deck
npm install
npm run build
```
